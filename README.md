<h1>Project Description</h1>
This repo trains a, Deep Reinforcement learning, agent to play the sega genesis game, 'Airstriker' through the tech stack of:
<ul>
  <li>OpenAI's gym retro library.</li>
  <li>Python.</li>
  <li>Tensorflow.</li>
</ul>
